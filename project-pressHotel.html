Press Hotel

prueba técnica servicios hoteleros.

Review

Hace parte de la presentación de una prueba técnica, donde se demostraron habilidades de maquetación y manejo de herramientas estándares en el desarrollo front end. Uso de lenguajes como HTML, CSS y JavaScript, conservando el estado puro de cada uno de ellos. Aplicación de sistema control de versiones con GIT, uso de patrón de maquetación Mostly Fluid y estrategias de desarrollo Mobile First son algunos de los conceptos apropiados en este proyecto.

  • Ver WebSite

Slider Principal

Al tener un sitio web que tomaba toda la pantalla con imágenes definidas, se genera un carrusel o slider que son uno de esos elementos web que permiten mostrar los productos y servicios de forma bastante llamativa en lo visual, este se crea a partir de html y estilos en estado puro de CSS.

  • Ver Detalle

Ubicación de Contenedores

La organización de contenedores se realiza a traves del modulo CSS Flexbox, con el cual se distribuye el espacio entre los ítems de la interfaz y asi se mejoran las capacidades de alineación, generando adaptación a los componentes utilizados y facilitando así el diseño responsivo.

  • Ver Detalle

Responsive Design

Diseño responsivo mediante el uso de técnicas con Breakpoints y Media Queries, dando como prioridad el desarrollo mobile basado en la estrategia Mobile First. Adicionalmente se utiliza el patrón de maquetación Mostly Fluid

  • Ver Detalle

Optimización de Imágenes

Optimización en cada una de las imágenes contenidas en el proyecto con el fin de favorecer el performance y/o rendimiento de la pagina web. Uso de herramientas dedicadas a la optimización. omitiendo metadatos innecesarios.

  • Metodología BEM (Block-Element-Modifier)
